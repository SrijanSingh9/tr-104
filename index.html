The Final Outcome: ILRE Project

Integrated Learning Recommendation Engine + Personalized PYLP Report Card. A decoupled dual-stage generative framework that transforms raw student data into actionable, AI-driven mentorship using Gemini API, Serper, and Streamlit.

JULY 2025

The Beginning

July 10 - 15

Orientation and setup. We dove straight into Python prerequisites and the foundations of Machine Learning. Assignments focused on data structures essential for future NLP tasks.

Core ML & Linear Regression

July 16 - 31

Intensive live sessions on Applied Machine Learning. Completed Coursework 1 (Linear Regression) and started diving into Logistic Regression. Daily routine involved 12 AM live sessions and heavy coding assignments.

AUGUST 2025

SQL & Data Structures

Aug 1 - 15

Shifted focus to Structured Query Language (SQL) and DSA. Understanding database management was crucial for the backend logic we would later build in MySQL.

Evaluations & Bayesian Learning

Aug 18 - 31

Milestone: Cleared the major Aptitude Test on Aug 18. Started advanced ML topics including Bayesian Learning and Hypothesis Testing.

SEPTEMBER 2025

Deep Learning & Dataiku

Sept 1 - 30

Transitioned into Deep Learning architectures. Simultaneously earned the Dataiku Core Designer Certification. The concepts of neural networks laid the groundwork for understanding LLMs.

OCTOBER 2025

NLP & Model Ops

Oct 1 - 31

Achieved Dataiku Advanced & MLOps Certifications. Deep dive into Natural Language Processing (NLP). We started analyzing text data, web scraping news articles, and understanding how machines process language.

NOVEMBER 2025: Project Inception

The "Tuesday & Thursday" Ritual

Weekly Mentorship

Every Tuesday and Thursday, we met with Sabudh mentors. We were assigned the Integrated Learning Recommendation Engine. Initial challenges included defining the scope: How to evaluate volunteers beyond just numbers?

R&D: Why not Standard RAG?

Mid-November

We explored standard RAG but realized vector DBs go stale. We pivoted to Indirect RAG using Serper API for live course fetching. Integrated Gemini API for semantic reasoning of student scores (Values, Competencies, Knowledge).

DECEMBER 2025

System Integration

Dec 1 - 15

Combined all modules. Built the Streamlit frontend with Multilingual support (Hindi/Punjabi). Implemented the "Check-or-Create" logic with MySQL to optimize API costs.

Final Polish

Dec 16 - 31

Final testing of the PDF generator (`ReportLab`). Generated documentation, user manuals, and the final project report. Prepared for the final defense.

JANUARY 6, 2026

Graduation Ceremony

Successfully presented the Integrated Learning Recommendation Engine. Graduated from the Sabudh Foundation Internship Program with advanced skills in Generative AI and Data Science.
